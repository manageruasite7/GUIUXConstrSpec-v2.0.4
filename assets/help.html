Программа GUIUXConstrSpec v2.0.4

Предназначена для визуального проектирования UI/UX-спецификаций.

1. Основная концепция

Программа позволяет создавать детальные спецификации, "обрисовывая" элементы интерфейса поверх загруженного изображения-прототипа. Каждый элемент в дереве слева связан с областью на холсте в центре и настраивается на панели справа.

2. Управление проектами

Используйте меню Файл для создания, открытия и сохранения проектов (.json). Для быстрого сохранения используйте Ctrl+S. Программа предупредит вас о несохраненных изменениях перед закрытием или созданием нового проекта.

3. Сценарии

Это глобальный инструмент для описания пользовательских сценариев (User Flows). Сценарии хранятся в отдельном файле scenarios.json и доступны для любого проекта. В текстовом редакторе вы можете описывать шаги пользователя, используя специальные ссылки на компоненты из любых ваших проектов-модулей, которые выбираются в окне "Вставить ссылку". Готовые сценарии можно экспортировать в .md файл через меню Сценарии -> Экспорт в Markdown...

4. Структура проекта (Левая панель)

  • Дерево компонентов: Здесь создается иерархия элементов вашего интерфейса (экраны, панели, кнопки).
  • Управление элементами: Используйте кнопки под деревом для добавления, удаления, перемещения и смены родителя у элементов.
  • Контекстное меню: Клик правой кнопкой мыши по элементу позволяет сохранить его как шаблон.
  • Шаблоны: Кнопка "+ Из шаблона" позволяет создавать новые элементы на основе ранее сохраненных. Управлять шаблонами (переименовывать, удалять) можно через кнопку "Управление шаблонами".

5. Холст (Центральная панель)

  • Фон: Для каждого элемента верхнего уровня ("экрана") можно загрузить свой фон-прототип (кнопка "Загрузить Прототип").
  • Привязка области: Создав элемент в дереве, выберите его и нажмите "Привязать/Изменить область", чтобы нарисовать для него прямоугольник на холсте.
  • Взаимодействие: Выделяйте элементы кликом, перемещайте их и изменяйте размер с помощью маркеров. Все изменения синхронизируются с панелью свойств.
  • Настройки: Кнопка "Настройки" открывает диалог, где можно задать внешний вид для элементов на холсте (вкладка "Внешний вид"), а также указать путь к папке с вашими проектами-модулями для функции "Вставить ссылку" (вкладка "Рабочая область").
  • Навигация: Зажмите среднюю кнопку мыши (колесико) для перемещения холста. Используйте Ctrl + колесо мыши для масштабирования.

6. Панель свойств (Правая панель)

  • Основные свойства: Задайте имя, тип и описание для выбранного элемента.
  • Таблица свойств: Добавляйте и редактируйте различные атрибуты элемента. Двойной клик по значению свойства цвета или шрифта откроет соответствующий диалог выбора.
  • События и Действия: Во второй вкладке вы можете описать логику работы элемента, добавляя на события (например, "Клик мыши") цепочки действий (например, "Открыть экран" или "Изменить свойство").
  • Глобальные сущности: Позволяют определять переменные (например, @user_name), которые можно использовать в параметрах действий.

7. Экспорт

Готовую спецификацию можно экспортировать в читаемый формат Markdown через меню Файл -> Экспорт в Markdown.... Полученный файл можно использовать как техническое задание для разработчиков или как подробный контекст для языковых моделей ИИ.
